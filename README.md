# RepoListExample
Example application for hexMachina framework. It loads a list of repositories from Github or Bitbucket and displays them.

*Find more information about hexMachina on [hexmachina.org](http://hexmachina.org/)*

## Compilation
Either use **compile.bat** to compile to js and neko or use the corresponding FlashDevelop project.
On Linux or Mac, just use the commands within the compile.bat.
